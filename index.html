The Danger Using em To Set Font Sizes

The use of the em unit in setting font size can cause some crazy cascading effects

  • This is a list item within the first ul
  • Its font size is set to 1.125em (about 18px)
  • I am 1em too
    • This is a list item in a second, nested ul
    • This is where the cascading em effect begins to be noticeable
    • I am now at 1.125em * 1.125em = 1.265 (about 20.25px)
      • This is a list item in the third, nested ul
      • The multiplicative effect continues
      • I am now at 1.125em * 1.125em * 1.125 = 1.4238 (about 22.78px)
